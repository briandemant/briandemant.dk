h1. iamneato.com

This is a simple but deceivingly powerful Sinatra blog. It gives you full control of your website and it's a great way to learn Sinatra.

The code for this blog originated from Hugh Evans but has been modified to fit my needs. You can find the original repository here "hughevans.net":http://github.com/artpop/hughevans.net/tree/master.
